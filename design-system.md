# Design System

## Visual Direction

- Modern premium Indian educational/business website
- Clean typography
- Strong whitespace usage
- Minimal gradients
- Elegant card layouts
- Professional not startup-gimmicky

## Typography

- Heading Font: Poppins
- Body Font: Inter
- Strong hierarchy
- Large readable sections

## Colors

Primary: #0F172A
Accent: #2563EB
Neutral backgrounds

## Layout Principles

- Max width: 1280px
- Consistent padding
- Alternating section backgrounds
- Clear CTA hierarchy

## Component Style

- Rounded-xl cards
- Soft shadows
- Minimal borders
- Smooth hover states

## UX Rules

- Mobile-first
- Fast loading
- Accessibility compliant
- Minimal clutter
